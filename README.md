# dx-sxgo

双休购，用消费支持守规矩的企业

我们想做的事情其实很简单，也很朴素：把那些愿意认真对待员工、愿意遵守规则的企业，尽量清楚地展示出来，让每一次消费都多一点判断，也多一点温度。

这个项目现在是一个轻量、可直接部署的 Next.js 站点，已经包含：

- 企业展示
- 行业筛选和搜索
- 手机与桌面自适应
- 赞助、联系开发者、社群和 GitHub 链接

## 志愿者招募

项目维护、运维安全、企业审核、志愿者

```bash
http://localhost:3000
```

## 本地运行

```bash
npm install
npm run dev
```

## 构建与部署

```bash
npm run build
npm start
```

如果你部署到 Vercel，直接导入仓库就能上线。

## 数据维护

企业数据集中保存在：

```text
lib/companies.json
```

更新企业时，直接修改这个文件即可。

## 项目结构

```text
app/                  页面和路由
lib/companies.json    企业数据
public/               静态资源
```

## 开源说明

这个仓库已经整理成适合公开发布的形态。

它不是一个炫技项目，更像是一个想认真把事做好的小站。我们会尽量保持它简单、清晰、可维护，也欢迎大家一起把它变得更完整。

如果你愿意参与，真的很欢迎你：

- 提一个 issue，告诉我们哪里还不够好
- 提一个 PR，修一个小问题也很好
- 分享一个更可靠的数据源
- 帮忙发现页面、文案或体验上的不顺手

每一个建议我们都会认真看。哪怕只是一个小修正，也可能帮这个项目少走很多弯路。
